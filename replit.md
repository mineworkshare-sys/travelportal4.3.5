# Tour Portal Theme

## Project Overview
A custom WordPress theme for tour booking businesses. Supports two operating modes:
- **Single Operator**: One company runs the site under a `site_owner` role
- **Multi-Partner**: Multiple tour companies (partners) each get their own company landing page

Booking is handled via TicketingHub integration. Multilingual support via Polylang.

## User Preferences
- **Never change the theme name.** It must always remain "Tour Portal" in style.css and all references.
- Always update the version number in style.css when making any fix or new feature. Before setting a version number, confirm the current live version with the user to avoid going backwards.
- Always add a changelog entry to CHANGELOG.md for every fix or new feature, at the top of the file.
